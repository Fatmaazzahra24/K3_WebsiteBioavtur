PERAN KEBIJAKAN

Direktur menetapkan arah dan persetujuan program K3.

Direktur berada pada tingkat tertinggi untuk menetapkan kebijakan, memberikan arahan, serta menyetujui program dan dokumen K3 perusahaan.

PENGAWASAN

P2K3 mengoordinasikan penerapan sistem manajemen K3.

Ketua dan Sekretaris P2K3 bertugas mengawasi, mengevaluasi, dan memastikan penerapan K3 berjalan sesuai kebutuhan operasional dan ketentuan perusahaan.

PELAKSANAAN

Manager HSSE dan Tim HSSE menjalankan program di lapangan.

Manager HSSE mengelola aspek keselamatan, kesehatan kerja, keamanan, dan lingkungan, dibantu Sekretaris HSSE serta Tim HSSE untuk program teknis hingga dokumentasi K3.
